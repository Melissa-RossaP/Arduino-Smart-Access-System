### Arduino Smart Access System

Sistema de controle de acesso desenvolvido utilizando Arduino UNO, RFID, Servo Motor, Display OLED, LEDs e Buzzer.

O projeto realiza a leitura de cartões ou chaveiros RFID previamente cadastrados. Quando um dispositivo autorizado é identificado, o acesso é liberado por meio do acionamento do servo motor, acompanhado de sinalização visual e sonora.

### Tecnologias Utilizadas:

Arduino UNO

C++

RFID (MFRC522)

Servo Motor

Display OLED (SSD1306)

LEDs

Buzzer

### Funcionalidades:

Autenticação por RFID;

Controle do Servo Motor;

Feedback visual através do Display OLED;

Sinalização sonora utilizando Buzzer;

Controle de LEDs indicadores de acesso.

Desenvolvido para fins acadêmicos e de aprendizado.
